# Sprint Time Manager

מערכת React קטנה לניהול זמן לפי ספרינטים.

## מה יש בפרויקט

- React + Vite
- תמיכה בעברית ו-RTL
- שמירה ב-LocalStorage בדפדפן
- יצירת ספרינט פעיל
- הוספת משימות יומיות
- בחירת תחום עבודה: WCM Desktop / Mobile / Cursor
- ניהול רשימת תחומי עבודה
- ניהול סוגי פעילות
- סיכום ספרינט מוכן להעתקה
- היסטוריית ספרינטים

## הרצה מקומית

```bash
npm install
npm run dev
```

אחרי ההרצה פותחים בכרום:

```text
http://localhost:5173
```

## הערה חשובה

המידע נשמר כרגע ב-LocalStorage של הדפדפן. כלומר הנתונים נשמרים באותו מחשב ובאותו דפדפן.
